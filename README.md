# Blinkit Marketing Performance Analysis 📊

## 📌 Project Overview
An end-to-end SQL analysis of marketing data to identify high-performing channels and optimize budget allocation. 

## 🛠️ Tech Stack
* **Database:** PostgreSQL (pgAdmin 4)
* **Concepts:** Aggregations, Subqueries, CASE Statements, Data Cleaning

## 🔍 Key Insights
* **Channel Efficiency:** Identified which channels drive the highest Click-Through Rate (CTR).
* **Budget Optimization:** Located "Budget Burner" campaigns with high spend but low ROAS.
* **Temporal Trends:** Analyzed revenue patterns by day of the week to suggest better budget timing.

## 🚀 How to Use
1. Run the `Blinkit_Marketing_Analysis.sql` script in any PostgreSQL environment.
2. Ensure the dataset matches the schema provided in the `CREATE TABLE` section.
